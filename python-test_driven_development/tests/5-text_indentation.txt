text_indentation module.

This module contains a function text_indentation that prints text
with 2 new lines after each of these characters: '.', '?', and ':'.

Usage:
    >>> text_indentation = __import__('5-text_indentation').text_indentation

    # Basic cases
    >>> text_indentation("Hello. How are you?")
    Hello.
    <BLANKLINE>
    How are you?
    <BLANKLINE>
    >>> text_indentation(" Hello. World? ")
    Hello.
    <BLANKLINE>
    World?
    <BLANKLINE>
    >>> text_indentation("One sentence.")
    One sentence.
    <BLANKLINE>

    # No special characters
    >>> text_indentation("Just a normal line")
    Just a normal line
    >>> text_indentation("   Normal line with spaces   ")
       Normal line with spaces   

    # Multiple punctuations
    >>> text_indentation("Start? Continue. End:")
    Start?
    <BLANKLINE>
    Continue.
    <BLANKLINE>
    End:
    <BLANKLINE>

    # Whitespace handling
    >>> text_indentation("   Hello.   World?   ")
    Hello.
    <BLANKLINE>
    World?
    <BLANKLINE>

    # Empty string
    >>> text_indentation("")

    # Error cases
    >>> text_indentation(123)
